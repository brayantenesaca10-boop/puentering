# Puente - Cost Breakdown & Options

## Price Point: $49

Puente is a visibility product, not a tech gadget. $49 positions it as something worth having, not something cheap to try. At $29 people ask "is this legit?" At $49 people think "this is a real product." Someone already confirmed $50 is nothing. Trust the signal.

## Option A: Full Launch (Premium Experience) - $1,500-2,200

Everything in the roadmap. Rings, packaging, app, legal, branding.

| Item | Cost | Necessary? |
|---|---|---|
| 100 NFC rings (zirconia ceramic, express ship) | $400-600 | Yes |
| 100 magnetic boxes (custom, express ship) | $100-150 | No |
| Domain name | $12 | Yes |
| Puente sound (Fiverr) | $30-50 | No |
| Apple Developer Account | $99 | No (web app works) |
| Google Play Developer | $25 | No (web app works) |
| Insert cards (Staples) | $10 | No |
| Seal stickers (StickerMule) | $30-50 | No |
| Launch event | $100-200 | No |
| Trademark search (attorney) | $300-500 | Not yet |
| LLC registration (CT) | $120 | Not yet |
| Trademark filing | $250-350 | Not yet |
| **Total** | **$1,500-2,200** | |

### Revenue
- 100 rings x $49 = $4,900
- COGS per ring (ring + box + card + sticker): ~$7-9
- Gross profit: ~$3,900-4,200
- Net after all costs: **$2,700-3,400 profit**

### Risk
- You spend $1,500-2,200 before selling a single ring
- If demand is weak, you're sitting on inventory + sunk costs
- Legal costs are front-loaded for something that may not need protection yet

## Option B: Lean Launch (MVP) - $450-650

Only what you NEED to sell rings. Cut everything that doesn't directly put a ring in someone's hand.

| Item | Cost | Notes |
|---|---|---|
| 100 NFC rings (zirconia ceramic, express ship) | $400-600 | The product. Non-negotiable. |
| Domain name | $12 | Need a URL for the rings to point to |
| Insert cards (print at home) | $0 | Print QR codes on paper you already have |
| **Total** | **$412-612** | |

### What you CUT and why
| Cut | Saves | Why it's OK |
|---|---|---|
| Magnetic boxes | $100-150 | Use small velvet drawstring bags from Amazon ($8 for 50). Still feels premium, fraction of the cost. |
| Custom sound | $30-50 | Use a stock sound effect for now. Replace later. |
| Apple Developer ($99) | $99 | Web app only for launch. Rings open puente.app in the browser. App comes after you validate demand. |
| Google Play ($25) | $25 | Same - web first. |
| Seal stickers | $30-50 | Skip for now. |
| Launch event | $100-200 | Sell at existing community events instead of hosting your own. Zero cost. |
| Trademark search | $300-500 | File later once revenue is flowing. Risk is low at 100 units. |
| LLC | $120 | Sell as sole proprietor for the first batch. Form LLC before batch 2. |
| Trademark filing | $250-350 | Same - after revenue. |

### Revenue
- 100 rings x $49 = $4,900
- COGS per ring (ring + bag): ~$4.50-6.50
- Gross profit: ~$4,250-4,450
- Net after all costs: **$4,250-4,450 profit**

### Risk
- No legal protection (low risk at this scale)
- No native app (web app works fine)
- Less premium unboxing (but velvet bag still feels nice)

## The Play: Option B Then Upgrade

**Batch 1: Option B - $450-650**
- 100 rings in velvet bags
- Web app only
- Sell at churches, soccer leagues, barbershops, schools
- Goal: sell all 100, pocket ~$4,350

**Batch 2: Upgrade to Option A with batch 1 profits**
- Order 200-500 rings (lower per-unit cost at scale)
- Add magnetic boxes with debossed logo
- File LLC + trademark
- Launch native app (Apple + Google)
- Custom Puente sound
- Host a real launch event
- You're now spending REVENUE not savings

This way you never risk more than $650 of your own money. Everything after that is funded by the business.
